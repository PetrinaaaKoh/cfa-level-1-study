# CFA Level I Study Materials

Comprehensive, interactive study platform for CFA Level I exam preparation.

## Features

- ✅ All 10 CFA Level I topics covered
- ✅ 200+ practice questions with detailed explanations
- ✅ Progress tracking (saved locally)
- ✅ Formula reference sheets
- ✅ Search functionality
- ✅ Mobile-responsive design
- ✅ Works offline after initial load

## Topics Covered

1. Ethical and Professional Standards (15-20%)
2. Quantitative Methods (8-12%)
3. Economics (8-12%)
4. Financial Statement Analysis (13-17%)
5. Corporate Finance (8-12%)
6. Equity Investments (10-12%)
7. Fixed Income (10-12%)
8. Derivatives (5-8%)
9. Alternative Investments (5-8%)
10. Portfolio Management (5-8%)

## Usage

Simply open `index.html` in a web browser, or access via GitHub Pages.

## Deployment

This is a static site. Deploy to GitHub Pages:

1. Push to GitHub repository
2. Go to Settings → Pages
3. Select main branch as source
4. Access via: `https://username.github.io/repo-name/`

## Local Development

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve
```

Then open `http://localhost:8000` in your browser.
